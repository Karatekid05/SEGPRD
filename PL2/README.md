# PL2 — Segurança e Privacidade de Dados

Practical Lab 2: PGP, RSA, Hashing Functions, File Integrity.

## Structure

| Folder | Description |
|--------|-------------|
| **Exercise1-PGP** | PGP: key pair, key server, email integration, encrypt/sign, backup design |
| **Exercise2-RSA** | RSA with OpenSSL: key pair, encrypt/decrypt file, ransomware/entropy notes |
| **Exercise3-Hashing** | Hash identification, cracking with John the Ripper/Hashcat, collisions |
| **Exercise4-FileIntegrity** | shasum, Shattered collisions, SHA-256 vs SHA-512, script + AIDE conclusions |

Each exercise has its own `README.md` with the full report and instructions.
